# [Organization] OSINT Investigation Methodology

## Phase 0: Authorization and Setup
- [ ] Scope of work signed
- [ ] Legal review completed
- [ ] OPSEC measures in place
- [ ] Investigation VM prepared
- [ ] Case file structure created
- [ ] Documentation template opened

## Phase 1: Requirements Definition
- [ ] PIR defined and documented
- [ ] SIRs and EEIs derived
- [ ] Source list identified
- [ ] Collection plan drafted
- [ ] Timeline established

## Phase 2: Initial Collection (Passive)
- [ ] People search engines queried
- [ ] Social media platforms searched
- [ ] Public records queried
- [ ] News archives searched
- [ ] Corporate registries checked
- [ ] Court records searched
- [ ] All results documented with timestamps

## Phase 3: Deep Collection (Active, if authorized)
- [ ] Infrastructure scanning (if authorized)
- [ ] Direct website visits
- [ ] Advanced search techniques (Google dorking)
- [ ] Platform-specific deep searches
- [ ] All results documented

## Phase 4: Processing
- [ ] Data normalized
- [ ] Duplicates removed
- [ ] Foreign-language content translated
- [ ] Metadata extracted
- [ ] Evidence hashed and preserved

## Phase 5: Analysis
- [ ] Timeline constructed
- [ ] Network map created
- [ ] Hypotheses developed and tested
- [ ] ACH or other SAT applied
- [ ] Key assumptions checked
- [ ] Confidence levels assigned

## Phase 6: Quality Control
- [ ] Peer review completed
- [ ] Bias check completed
- [ ] All sources verified
- [ ] Evidence chain documented
- [ ] Findings cross-referenced

## Phase 7: Reporting and Dissemination
- [ ] Report drafted
- [ ] Report reviewed
- [ ] Report delivered to consumer
- [ ] Feedback solicited

## Phase 8: Case Closure
- [ ] Case files archived
- [ ] Retention schedule applied
- [ ] Lessons learned documented
- [ ] Methodology updates identified
