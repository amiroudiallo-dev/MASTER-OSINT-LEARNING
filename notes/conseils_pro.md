**Conseils de pro**
 + Écris tes exigences avant de toucher un clavier. L’erreur la plus courante dans l’OSINT est de commencer à collecter avant de définir ce que vous recherchez.
 + Maintenez une matrice de fiabilité de source. Toutes les sources ne se valent pas. Suivez quelles sources ont historiquement fourni des informations exactes.
 + Chronomètre ta collection. Fixez une limite de temps pour chaque phase de collecte. Il est toujours possible d’en collecter davantage ; La question est de savoir si une collecte supplémentaire modifiera vos conclusions analytiques.
 + Version de votre analyse. À mesure que de nouvelles informations arrivent, vos évaluations peuvent évoluer. Conservez l’historique des versions afin de pouvoir retracer l’évolution de vos conclusions.
 + Séparer le fait de l’inférence. Dans vos produits finis, distinguez clairement ce que vous avez observé (fait) de ce que vous avez conclu (inférence).
