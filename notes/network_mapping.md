## Analyse des liens (cartographie réseau)

Cartographier les relations entre entités pour identifier les nœuds clés, les gardiens et les connexions cachées.

### Outils pour l’analyse des liens :

  [-]Maltego : outil commercial pour l’analyse graphique des liens avec des transformations automatisant la collecte de données.
  [-]Gephi : plateforme open source de visualisation de graphes.
  [-]i2 Analyst's Notebook : outil d'analyse de liens policiers standard dans l'industrie.
  [-]Obsidian avec vue graphique : option légère pour les enquêtes plus petites.
